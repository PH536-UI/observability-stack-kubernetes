# Observability Stack

Stack de observabilidade utilizando:

- Grafana
- Loki
- Promtail
- Docker Compose

## Funcionalidades

- Coleta de logs Linux
- Coleta de logs Docker
- Centralização em Loki
- Visualização em Grafana

## Executar

```bash
docker compose up -d
```

## Acessar

Grafana:

http://localhost:3000
